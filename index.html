HealthCodec

Healthcodec é um formato compacto, rápido e eficiente para armazenar informação em saúde registrada em objetos do modelo de referência do openEHR.

Google faz uso de Protocol Buffers e FlatBuffers, Netflix usa Hollow, Github usa BERT, e Redis usa MessagePack para serialização de dados em geral, dentre outras opções. Healthcode é uma estratégia específica para informação em saúde, e mais eficiente.
